Keyboard shortcuts which are recomended by our coding sensei Valdis

Ctrl+Shift+Arrow up or arrow down | applicable to code editor | Allows you to copy the entire row up or down.

Cntr+D(when a word is highlighted) | applicable to code editor | Lets you rename multiple words at once.

Shift+Tab | applicable to code editor | Use like Tab's reverse

Ctrl+F | applicable in web browsers | To find a specific word or multiple words.

Ctrl+shift+T | applicable in web browsers | Reopen closed Tab in web browser.
